Retrosynthesis — "how would you make it?"

Real ML retrosynthesis (AiZynthFinder, ASKCOS, IBM RXN) predicts multi-step routes back to purchasable building blocks from millions of known reactions. It's legitimate and chemists use it — but it needs a backend (a Python service), it's organic-molecules only, and every route is a suggestion requiring lab validation, never a guarantee. So this page is a client: point it at a retrosynthesis backend and it'll show routes; with none connected it makes zero external calls and just explains the honest setup.

Expected: POST {smiles} → {routes:[{steps:[...], score}]}. Nothing is sent anywhere unless you set this to your own endpoint.

Stand up a free backend (AiZynthFinder on a Hugging Face Space)

  1. Create a free HF Space (Docker or Gradio).
  2. pip install aizynthfinder and download its public models/stock (download_public_data).
  3. Wrap aizynthfinder in a tiny FastAPI/Gradio endpoint that accepts a SMILES and returns the top routes as JSON.
  4. Paste the Space URL above. (ASKCOS and IBM RXN also expose APIs you can adapt.)
